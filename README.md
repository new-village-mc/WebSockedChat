# WebSockedChat

Это просто мост между чатом майнкрафта и чем-либо еще. Работает на WebSocket

```
ws://127.0.0.1:8080
```

формат данных 
обычное сообщение:
```
{
    "type": "message", 
    "data": {
        "text": "test text", // текст
        "author": "Evgeniy1357" // отправитель
    }
}
```
Сообщение из вебкарты
```
{
    "type": "webmap", 
    "data": {
        "text": "test text"
    }
}
```
